# 프로젝트 이름

## 프로젝트 설명
웹 프로젝트를 새로시작할때에 대한 대략적인 틀 저장소입니다.
폴더구조는 기본 권장사항으로 구성이되어있으며 필요에따라 크게 바뀌는경우는 
새로운 저장소 또는 브랜치로 관리하여 관리에 용이하고자 합니다.
